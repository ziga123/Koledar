# Koledar

Namizna aplikacija za prikazovanje koledarja, napisana v Pythonu, pri čemer je za GUI uporabljena knjižnica CustomTkinter.

## Funkcionalnosti
- Skok na izbran mesec v izbranem letu
- Vnos natančnega datuma in skok na ustrezni mesec v ustreznem letu
- Nedelje so obarvane z rdečo
- Prazniki, prebrani iz datoteke Prazniki.txt, so obarvani z vijolično (tudi, če pade na nedeljo)

## Zagon
1. (Na Windows) Zaženi Koledar.exe v mapi dist
2. Iz mape src zaženi skripto main.py, pri čemer mora biti inštaliran modul CustomTkinter
